## â“ Common Questions

### Q: Will I lose my data if I clear browser cache?
**A:** Yes! Export it first (`Export Data` button). Or don't clear cache. Data lives in localStorage.

### Q: Can I use on multiple devices?
**A:** Yes! Use GitHub Pages for online access from anywhere. Or export/import data between devices.

### Q: Is my attendance data private?
**A:** Yes! Data stored only on your device. Never sent anywhere. Completely private.

### Q: What if I miss marking attendance?
**A:** Edit anytime! Go to Calendar, click class, change the mark. No time limits.

### Q: Can I change my timetable mid-semester?
**A:** Yes! Edit or delete timetable entries anytime. Attendance records stay.

### Q: What happens at semester end?
**A:** Export your data for records. Clear data and start fresh, or keep it. Your choice.

### Q: Does it work offline?
**A:** Yes! Local version works offline. GitHub Pages needs internet, but then works offline.

### Q: Can I change the color scheme?
**A:** Yes! Subject colors are customizable. Overall theme in `css/styles.css`.

## ğŸ› ï¸ Troubleshooting

### Data Not Saving?
- Check browser localStorage is enabled
- Try different browser
- Check you're connected (if using GitHub Pages)

### Can't Mark Attendance?
- Make sure timetable has classes for that day
- Try clicking different parts of the dot
- Refresh and try again

### Predictions Not Showing?
- Add timetable entries (system needs to know class schedule)
- Refresh the page
- Check you have at least 1 subject and 1 class

### App Running Slow?
- Clear browser cache
- Close other browser tabs
- Try different browser
- Update your browser

### Lost Data?
- Import from backup (if you exported)
- You might have had multiple browsers with different data
- Data is per-browser on local version
